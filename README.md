# 🏧 ATM Interface (Java)

A simple Java-based ATM Interface developed using Object-Oriented Programming (OOP). This project simulates basic ATM operations such as deposit, withdrawal, balance inquiry, fast cash, and transaction history.

## 📌 Features

- 🔐 Secure PIN Authentication (3 Attempts)
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Account Balance
- ⚡ Fast Cash (INR 500, 1000, 2000, 5000)
- 🧾 Transaction History
- ✅ Input Validation
- 🚪 Exit Option

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- ArrayList

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ATM-Interface.git
```

2. Open the project in VS Code, IntelliJ IDEA, or Eclipse.

3. Compile the Java files:

```bash
javac *.java
```

4. Run the project:

```bash
java Main
```
